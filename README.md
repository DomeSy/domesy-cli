# domesy
通过 cli 创建模板

- Ant Design Pro V5（PC端)模板： https://github.com/DomeSy/Ant-Design-Pro-V5
- ant-mobile(移动端)模板：https://github.com/DomeSy/react-mobile

## 安装
```bash
  # npm
  $ npm install domesy-cli -g

  # yarn
  $ yarn global add domesy-cli
```
